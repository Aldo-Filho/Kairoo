# 🎮 Kairoo

## 📌 Descrição
O **Kairoo** é um site desenvolvido em PHP como parte do Projeto Integrador da turma Mediotec Senac (2º ano - 2025).

A plataforma foi criada com o objetivo de apresentar o projeto principal e servir como base para hospedar um jogo educacional 2D.

---

## 🎯 Objetivo do Projeto
O sistema funciona como uma interface entre professores e alunos, onde:

- Professores podem criar perguntas dentro da plataforma
- Alunos acessam o sistema e utilizam essas perguntas dentro do jogo

O jogo (ainda não finalizado) consiste em um personagem que percorre um mapa e enfrenta inimigos respondendo perguntas criadas pelos professores.

## ⚠️ Observações Importantes
- A modelagem do banco de dados não está presente neste repositório, pois foi desenvolvida por outro integrante da equipe.
- O jogo 2D ainda não está integrado à plataforma, pois não foi finalizado.
- O site funciona como base estrutural para futura integração do jogo.

---

## ▶️ Como executar

1. Instale um ambiente servidor local (ex: XAMPP ou WAMP)
2. Coloque os arquivos do projeto na pasta `htdocs` para XAMPP, ou `www` para WAMP
3. Configure o banco de dados no phpMyAdmin
4. Inicie o Apache e MySQL
5. Acesse no navegador:
```
http://localhost/kairoo/public
```
